Sistema avançado de flashcards com repetição espaçada (FSRS), integrado ao Tutor LMS.

== Description ==

FSRS Flashcards for Tutor LMS adiciona ao WordPress um sistema profissional de flashcards
baseado no algoritmo FSRS (Free Spaced Repetition Scheduler), utilizado por ferramentas
como o Anki.

O plugin é totalmente integrado ao Tutor LMS, permitindo que alunos estudem flashcards
associados aos cursos, com controle de revisões, estatísticas e alto desempenho.

Principais recursos:

* Criação de decks e flashcards
* Integração nativa com Tutor LMS
* Modo de estudo com avaliações (again, hard, good, easy)
* Algoritmo FSRS para cálculo de revisões
* Banco de dados otimizado com tabelas próprias
* Painel administrativo com estatísticas e logs
* Arquitetura escalável para alto volume de usuários

== Installation ==

1. Envie a pasta `fsrs-flashcards-tutor` para o diretório `/wp-content/plugins/`
2. Ative o plugin no menu "Plugins" do WordPress
3. Certifique-se de que o Tutor LMS está instalado e ativo

== Frequently Asked Questions ==

= Este plugin funciona sem o Tutor LMS? =
Não. O plugin foi projetado para integração direta com o Tutor LMS.

= Os dados são armazenados no banco do WordPress? =
Sim. O plugin utiliza tabelas próprias para garantir performance e escalabilidade.
